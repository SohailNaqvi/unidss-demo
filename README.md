# University Decision Support System — Demo

A read-only preview of the University DSS prototype.

**Live demo:** Once deployed via GitHub Pages, this repo serves the `index.html` file at the root.

## Local preview

```bash
open index.html
```

## Deployment (GitHub Pages)

1. Push this folder to GitHub as a new repo.
2. In the repo settings → Pages → Source: `Deploy from a branch` → Branch: `main` / root.
3. Wait ~1 minute for the URL to go live.

The site is a single static HTML file — no backend required.
